[타이틀]
스페이스 바로 로비 진입 가능
1을 누르면 맵 제작 툴 진입 가능

[로비]
스페이스 바 / 전투 시작 버튼 눌러서 게임 진입 가능
현재 로비 기능 없음

[게임]
게임에서 입력 키
F4 전투 종료 (초기화)
F5 다음 스테이지
F6 이전 스테이지
F7 히트박스 on/off
숫자1 돈 +100
드래그로 화면 위 아래 이동 가능.

모든 화면에서 esc를 누르면 이전 화면으로 돌아감

영웅 뽑기 가격 3원
장비 뽑기 가격 5원
같은 등급, 같은 종류의 영웅으로 드래그 시 7성까지 합성 가능
같은 등급, 같은 종류의 장비로 드래그 시 4성까지 합성 가능
우클릭시 영웅/장비 판매 가능. 영웅은 원가의 2/3, 장비는 원가의 3/5
아이템
검(+공격력), 지팡이(*주문력), 활(+공격속도), 갑옷(시작시 쉴드추가)

스킬은 없는 버전.
원거리 공격이 투사체가 없음.
백그라운드의 타일이 좌우가 반전 되어있음
유닛 배치가 안된다면, 전투 중인 상태이니 F4를 누르세요
드래그, 전투 시작 시 화면이 벗어나는 버그가 간혹 일어남

[제작 툴]

몬스터/장애물 - 1~3 챕터까지 생성 가능, 1챕터 1~10 스테이지 만들면 게임 가능
백그라운드 - 10 x 15에 자유롭게 타일 배치가능
둘다 save/load 가능 (Json형식)
